## Kinematic Equations
$$
\bar{v} = \frac{v_i+v_f}{2}
$$
$$
v_f = v_i + at
$$
$$
\Delta x = v_i t + \frac{1}{2}at^2
$$
$$
v_f^2 = v_i^2 + 2a \cdot \Delta x
$$
**NOTE:** These equations are valid only when the acceleration $(a)$ is *constant*.
### Strategies For Solving Kinematics Problems
- Initial Velocity $(v_i)$
- Final Velocity $(v_f)$
- Acceleration $(a)$
- Time $(t)$
- Distance Travelled $(\Delta x)$
	- Initial Position $(x_i)$
	- Final Position $(x_f)$

There are only 5 variables you need to think of when solving kinematics problems (or 6 if initial and final positions are given differently).

You need to read the questions and find these values in the question. Then you are going to use these values to find the missing values by using kinematic equations above.

Also, for 2D kinematics problems you need to separate the question into horizontal and vertical parts. That way you can solve each of them separately. Don't forget that the *time $(t)$* will be the same for each of the parts.
